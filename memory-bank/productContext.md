# Product Context

## Why This Project Exists

v2rayN-Go 是对原版 v2rayN（C#/.NET）的 Go 语言重写，目标是：
- 去除 .NET 运行时依赖，实现单文件跨平台分发
- 提供 Web UI 替代桌面 GUI，支持远程管理
- 保持与原版兼容的协议支持和配置格式

## Problems It Solves

1. **跨平台部署** — 单一二进制，无需安装运行时
2. **远程管理** — 通过 Web UI 在任何设备上管理代理节点
3. **多协议支持** — 10 种主流代理协议统一管理
4. **多内核切换** — 支持 Xray/Sing-box/Mihomo 按协议自动选择最优内核
5. **订阅管理** — 支持订阅链接自动更新、正则过滤、批量导入
6. **节点测速** — 内置延迟测试，支持单节点和批量测速

## How It Should Work

1. 用户运行二进制 → 自动初始化 SQLite + 创建默认分组
2. Web UI 在 `127.0.0.1:2017` 启动
3. 用户通过 Web UI 添加节点（手动/订阅链接/二维码图片导入）
4. 选择节点 → 启动内核 → 系统代理自动配置
5. 实时查看连接状态、流量统计、内核日志

## User Experience Goals

- 简洁直观的 Web 界面，无需学习成本
- 拖拽排序节点和分组
- 一键测速、批量去重
- 多语言支持（中/英文）
- 深色/浅色/跟随系统主题
